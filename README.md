### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/chelseaharris137/Site-Mockup.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the app in the development mode.
   ```JS
    npm run dev
   ```
4. Open http://localhost:3000 to view it in the browser.
